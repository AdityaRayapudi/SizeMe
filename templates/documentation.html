Windows Installation

To install SizeMe on Windows:

python -m venv env
source ./env/scripts/activate
pip install -r requirements.txt
flask run

Mac OS / Linux Installation

To install SizeMe on macOS or Linux:

python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
flask run
